# أصول المتجر (AppGallery)

أضف هذه الصور قبل بناء الإصدار:

| الملف | المقاس | ملاحظة |
|-------|--------|--------|
| `icon.png` | 1024×1024 | أيقونة التطبيق |
| `adaptive-icon.png` | 1024×1024 | أيقونة أندرويد (شعار بخلفية شفافة) |
| `splash.png` | 1284×2778 | شاشة البداية (اختياري) |

بدونها قد يفشل `prebuild` — استخدم شعار قسطاس بخلفية برتقالية `#E85D04`.

لقطات شاشة للمتجر (هواوي): 3–5 صور من التطبيق على جوال حقيقي.
